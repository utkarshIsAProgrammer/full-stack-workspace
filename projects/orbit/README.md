# Orbit

Social media platform — **REST API backend** for posts, comments, follows, likes, reposts, saves, and notifications.

## Documentation

| File                                             | Description                                           |
| ------------------------------------------------ | ----------------------------------------------------- |
| [`backend/README.md`](backend/README.md)         | Setup, env vars, scripts, quick API overview          |
| [`PROJECT.md`](PROJECT.md)                       | Full technical reference (routes, models, middleware) |
| [`FEATURES.md`](FEATURES.md)                     | How each feature works (plain language)               |
| [`FRONTEND_AI_PROMPT.md`](FRONTEND_AI_PROMPT.md) | Copy-paste prompt to build a matching frontend        |

## Quick start

```bash
cd backend
cp .env.example .env   # fill in all values; CLIENT_URL must be a full URL
npm install
npm run dev
```

API base: `http://localhost:5000/api` — use `credentials: "include"` for cookie auth.

## Project layout

```
orbit/
├── backend/          # Express + TypeScript API
├── PROJECT.md
├── FEATURES.md
└── FRONTEND_AI_PROMPT.md
```
